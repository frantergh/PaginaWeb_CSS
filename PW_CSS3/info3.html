Software de Codificacion

Listado de FreeSoftware

SublimeText3: Sublime 3 es un editor de código muy rápido y sofisticado que nos permitirá ejecutar un montón de tareas de forma muy rápida y sencilla. Soporte muchos lenguajes de programación y dispone de un sistema de instalación de paquetes adiciones que amplían sus características de forma ilimitada.

Notepad++: Editor de texto y de código fuente libre con soporte para varios lenguajes de programación. Con soporte nativo para Microsoft Windows. Se parece al Bloc de notas en cuanto al hecho de que puede editar texto sin formato y de forma simple.

Brackets: Es un editor de texto de código abierto que se ha escrito en JavaScript, así como en HTML y CSS, con el objetivo de desarrollo web firmemente en mente. Esto está diseñado para simplificar el proceso de codificación, mientras que los programadores podrán compartir su trabajo a través de diferentes plataformas y aprovechar las aplicaciones de código abierto.

Visual Studio Code: Visual Studio Code es un editor de código fuente desarrollado por Microsoft para Windows, Linux y macOS. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código

Conclusion:

Cada uno puede utilizar este gran listado de herramientas de codigo abierto para codificar a su gusto y semejanza, todos ellos sirven para lo mismo, pero cada persona se siente mas comod en un entorno que en otros, cada software tiene diferentes herramientas y de mas.
